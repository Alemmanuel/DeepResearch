### Your First Research Query

```python
from deepresearch import ResearchEngine

# Initialize the beast
engine = ResearchEngine()

# Ask anything. Literally anything.
results = engine.research(
    query="What are the implications of quantum computing on cryptography?",
    depth="maximum",
    sources=["web", "academic", "internal_db"]
)

# Witness the magic
print(results.synthesized_conclusion)
print(f"Confidence: {results.confidence}%")
print(f"Sources analyzed: {len(results.sources)}")
```
